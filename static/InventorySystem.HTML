InventorySys MVP

Introduction

Project Overview & Scope

ระบบจัดการคลังสินค้า (Inventory Management System) รูปแบบ MVP พัฒนาขึ้นเพื่อแก้ปัญหาการจัดการสต็อก การบันทึกรับเข้า-จ่ายออก และการออกเอกสารใบเสร็จอย่างง่าย มุ่งเน้นความรวดเร็ว แม่นยำ และใช้งานง่ายบนระบบเครือข่ายภายใน

🐍

Backend

Python 3.12 + Flask Framework

🎨

Frontend

HTML5, Bootstrap 5 (Lux Theme)

🗄️

Database

SQLite (Relational File-based)

🌐

Environment

Localhost / Intranet Web App

⚠️ System Constraints (ข้อจำกัดระบบ)

  • • Database: SQLite ไฟล์เดี่ยว เหมาะสำหรับข้อมูลขนาดเล็ก-กลาง (Concurrency ต่ำ)
  • • Connectivity: ต้องรันบน Server ที่มี Python ติดตั้ง (On-Premise)
  • • Storage: เก็บรูปภาพใน Local Filesystem (ไม่ใช่ Cloud)

Architecture & Schema

ERD Database Schema (SQLite)

Hover over tables to visualize relationships (Foreign Keys).

User id (PK)
  • username VARCHAR
  • password VARCHAR
  • role VARCHAR
Product id (PK)
  • sku, name VARCHAR
  • current_quantity INTEGER
  • cost_price, selling_price FLOAT
  • min_quantity INTEGER
Transaction id (PK)
  • Core Data
  • type VARCHAR (IN/OUT)
  • quantity INTEGER
  • timestamp DATETIME
  • Relationships (Hover me)
  • sku FK (Loose)
  • party_name (Customer) Link
  • party_name (Supplier) Link
Customer id (PK)
  • name VARCHAR
  • phone, address VARCHAR
SystemConfig id (PK)
  • key VARCHAR (Unique)
  • value TEXT

Documentation

User Manual & Roadmap

1. การเริ่มระบบ

  • เข้าสู่ระบบ: User admin / Pass 1234

2. การจัดการ

  • Manage Users: เพิ่ม/ลบพนักงาน รีเซ็ตรหัสผ่าน
  • Settings: ตั้งค่าชื่อบริษัท ที่อยู่ โลโก้
  • Product: แก้ไขราคาทุน/ราคาขาย ลบสินค้า

1. จัดการสินค้า

  • Stock In: กดปุ่ม "+" -> เลือก In -> ระบุจำนวน -> เลือก Supplier
  • Stock Out: กดปุ่ม "+" -> เลือก Out -> ระบุจำนวน -> เลือกลูกค้า

2. เอกสาร

  • ไปที่เมนู History
  • กดไอคอนเครื่องพิมพ์เพื่อออก Invoice/Delivery Note
